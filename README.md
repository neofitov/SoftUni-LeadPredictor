# LeadPredictor

A sales funnel calculator that helps you estimate the number of prospects, leads, and customers needed to achieve your revenue goals.

## Features

- Configure campaign dates, revenue targets, and average order value
- Adjust lead and prospect response rates with interactive sliders
- Visualize monthly projections with a horizontal bar chart
- Language and currency selector

## Formulas

- **Customers** = Total Revenue ÷ Avg Order Value
- **Leads** = Customers × 100 ÷ Lead Response Rate (%)
- **Prospects** = Leads × 100 ÷ Prospect Response Rate (%)
- Monthly values are cumulative across the campaign duration
